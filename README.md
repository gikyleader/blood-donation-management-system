# Blood Donation Management System 🩸

## 📌 Overview
This project is a DBMS-based system designed to manage blood donation, storage, and hospital requests efficiently.

## 🔹 Features
- Donor Registration & Management
- Blood Inventory Tracking
- Hospital Blood Requests
- Critical Request Monitoring

## 🔹 Database Concepts Used
- ER Model (7 Entities)
- Relational Schema
- Normalization (3NF)
- SQL Queries (JOIN, GROUP BY, Aggregation)

## 🔹 Key Highlights
- Implemented real-world healthcare database
- Used JOIN operations to connect multiple tables
- Applied aggregation queries for analysis

## 🚀 Future Scope
- Web-based system
- Real-time alerts
